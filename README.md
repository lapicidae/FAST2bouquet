# FAST2bouquet


:warning: *Under development. Use with caution.* :warning:


**F**ree **A**d-Supported **S**treaming **T**elevision (FAST) Enigma2 bouquet generator. 

This tool automates the creation of IPTV bouquets and EPG configurations for Enigma2 based receivers, utilizing the processed data and XMLTV feeds from [Matt Huisman](https://i.mjh.nz/~~SUPPORT_ME~~).

Also, it is possible to generate a M3U playlist which is compatible with matthuisman's xmltv data for other players.
