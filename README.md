# FAST2bouquet

:warning: *Under development. Use with caution.* :warning:

**F**ree **A**d-Supported **S**treaming **T**elevision (FAST) Enigma2 bouquet generator. 

This tool automates the creation of IPTV bouquets and EPG configurations for Enigma2-based receivers (e.g. Dreambox, VU+).

## Key Features
* **Multi-Provider Support**: Some of the well-known FAST providers are supported
* **Multi-region & Language Support:** Fetch channels for different regions (DE, AT, CH, UK, US, etc.) depending on the provider.
* **Picon Download:** Download and automatic post-processing (rounded corners).
* **EPG Integration:** Generates XMLTV-compatible source files for plugin EPGImport.
* **M3U Export:** Can create independent M3U playlists that are compatible with the XMLTV sources listed below.
